# Mahi Paint House — Website

A responsive business website for **Mahi Paint House**, a paint and hardware store located in Bijainagar, Rajasthan.

🔗 **Live Site:** [Add your GitHub Pages link here, e.g. https://sakshihub-96.github.io/mahi-paint-house/]

## About

This site was built to give Mahi Paint House an online presence — showcasing the business, its products/services, and contact details for local customers.

## Features

- Clean, responsive design that works on mobile and desktop
- Business information and contact details
- Fast-loading, lightweight static site (no backend required)

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Project Structure

```
mahi-paint-house/
├── index.html      # Main page
├── style.css        # Styling
├── script.js         # Interactivity
└── files.zip         # Additional project assets
```

## Getting Started

To run this project locally:

1. Clone the repository
   ```
   git clone https://github.com/Sakshihub-96/mahi-paint-house.git
   ```
2. Open `index.html` in your browser

No build tools or dependencies required.
